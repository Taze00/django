Mein Ansatz

Ich glaube an Code der funktioniert, Interfaces die überraschen, und Produkte die Spaß machen zu benutzen.
  • 01

    Problem First

    Ich fange nie mit Code an. Erst verstehen was wirklich gebraucht wird, dann bauen.

  • 02

    Clean über Clever

    Lesbarer, wartbarer Code schlägt jeden Trick. Komplexität entsteht von alleine.

  • 03

    Design ist Funktion

    UX ist kein Nice-to-have. Wie etwas aussieht bestimmt wie gut es benutzt wird.

  • 04

    Immer am Lernen

    Jedes Projekt ist eine Chance, eine neue Technologie oder Pattern auszuprobieren.
